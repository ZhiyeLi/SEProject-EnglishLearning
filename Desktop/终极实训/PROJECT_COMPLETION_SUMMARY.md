## 🏗️ 技术架构

### 数据管理层

```
CheckinProgressManager (Singleton)
    ↓
localStorage
    ↓
JSON 序列化
```

**特点：**
- 单例模式 - 全局唯一实例
- 自动持久化 - 每次操作自动保存
- 错误恢复 - 数据损坏时重新初始化
- 轻量级 - < 1KB 占用

### 视图层

```
WordCheckin.vue
    ├── 状态管理 (Composition API)
    ├── UI 组件 (Scoped CSS)
    └── 事件处理 (emit/provide)
```

### 数据流

```
用户操作
    ↓
Vue 组件方法
    ↓
CheckinProgressManager 
    ↓
localStorage 保存
    ↓
UI 更新显示
```

## 🚀 使用指南

### 快速开始

```bash
# 1. 启动应用
npm run dev

# 2. 打开浏览器
http://localhost:5173

# 3. 进入打卡
主页 → 背单词 → 选择等级 → 打卡页面
```

### 核心功能

```javascript
// 1. 获取进度
checkinManager.getProgress('cet4')

// 2. 增加进度
checkinManager.incrementProgress('cet4')

// 3. 暂停
checkinManager.pauseCheckin('cet4')

// 4. 继续
checkinManager.resumeCheckin('cet4')

// 5. 切换等级
checkinManager.switchLevel('cet4', 'cet6')

// 6. 重置
checkinManager.resetProgress('cet4')
```
