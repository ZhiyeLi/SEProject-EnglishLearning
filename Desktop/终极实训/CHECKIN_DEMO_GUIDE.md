**步骤1：点击"开始打卡"**
```

```

**预期结果：**

### 演示场景 3：暂停功能

**步骤1：点击"暂停"按钮**
```
点击暂停 → 进度保持 5 → 按钮变化
```

**预期结果：**
```
按钮变化：   [继续打卡] [暂停] ────→ [继续打卡]
状态提示：   「正在进行中...」────→「已暂停·上次进度: 5」
进度保持：   进度条和进度文本不变
```

**步骤2：刷新浏览器**
```
关键验证：数据是否保存到 localStorage
```

**预期结果：**
```
刷新后显示：
进度文本：   5 / 3500
按钮：       [继续打卡]
状态提示：   「已暂停·上次进度: 5」
（表示数据成功持久化）
```

**步骤3：点击"继续打卡"**
```
继续打卡 → 进度 +1 → 界面恢复进行状态
```

**预期结果：**
```
进度文本：   6 / 3500
按钮：       [继续打卡] [暂停]
状态提示：   「正在进行中...」
```
