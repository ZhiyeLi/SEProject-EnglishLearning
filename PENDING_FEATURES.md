- 实现单词卡片展示（英文、音标、中文翻译）
- 实现"已掌握"和"不会"两个操作按钮

**涉及文件**：
- `src/views/` - 新增 `WordCheckInDetail.vue` 或类似组件

**依赖**：
- 需要依赖 `wordData.js` 的 `markWordAsPassed()` 方法更新进度

## 二、数据存储优化

### 1. 后端 API 集成（🔴 未开始）
**描述**：将本地 localStorage 数据同步到后端服务器

**需求**：
- 实现用户登录/注册功能
- 创建服务端 API 端点（详见 `API_INTERFACES.md`）
- 实现数据同步机制（手动/自动）
- 支持多设备数据同步
- 数据加密传输

**涉及文件**：
- `src/utils/` - 新增 `api.js` 或 `services/` 目录
- 后端服务（Node.js/Python/Java）

**API 端点**（参考 `API_INTERFACES.md`）：
- POST `/api/user/register` - 用户注册
- POST `/api/user/login` - 用户登录
- POST `/api/progress/save` - 保存学习进度
- GET `/api/progress/:typeId` - 获取学习进度
- POST `/api/plan/create` - 创建打卡计划
- POST `/api/plan/update` - 更新打卡计划
